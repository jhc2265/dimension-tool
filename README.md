# 📐 치수 표시 도구 (Dimension Annotation Tool)

사진 위에 치수선(화살표·눈금·라벨)을 그려 넣는 웹 도구입니다. 브라우저에서 바로 열려요 — 설치·인터넷 연결이 필요 없고, 프리텐다드 폰트가 파일에 내장되어 있습니다.

## 기능
- 사진 업로드 후 두 점을 이어 치수선 그리기 (화살표 / 눈금 / 둘 다 / 없음)
- 선을 선택하면 뜨는 빠른 편집 바에서 라벨·색상·두께 바로 수정
- 색상 팔레트 저장(직접 색 추가·우클릭 삭제), 라벨 배경, Shift로 각도 고정
- 되돌리기·전체 삭제, 확대/축소
- PNG · JPG · WebP로 저장 (배경 투명/색상 지정)

## 사용법
`dimension.html`을 브라우저로 열면 됩니다. GitHub Pages로 배포하면 `https://<아이디>.github.io/<저장소>/dimension.html` 주소로 바로 사용할 수 있어요.
